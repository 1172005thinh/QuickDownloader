# README

This **docs/** directory contains documentation for the project. It includes:

- **README.md**: This file, providing an overview of the documentation and its contents.
- **PROMPT.md**: A detailed command prompt for AI interactions of building the project.
- **REPORT.md**: A comprehensive report of what was done in the last prompt, including all the details of the implementation, fixing, and modification.
- **CHANGELOG.md**: A log of all the changes made to the project, including new features, bug fixes, and improvements.
- **project_guide/**: A directory containing guides and specifications for the project, including:
  - **ARCHITECTURE.md**: A detailed description of the project's architecture, including diagrams and explanations of the components and their interactions.
  - **API.md**: Documentation of the project's API, including endpoints, request/response formats, and examples.
  - **UI_UX.md**: A guide to the project's user interface and user experience design, including wireframes, mockups, and design principles.
  - **TEST.md**: A guide to the project's testing strategy, including test cases, testing tools, and best practices.
  - **PROJECT.md**: An overview of the project, including its goals, scope, key features and requirements.
